# GitHub Repository Setup Guide

Step-by-step instructions for configuring the HypoTestX GitHub repository
with the correct topic tags, description, and website URL.

---

## 1. Add Repository Description

1. Go to **https://github.com/Ankit-Anand123/HypoTestX**
2. Click the **gear icon (⚙)** next to "About" on the right-hand sidebar
3. In the **"Description"** field, enter exactly:

   ```
   Natural language hypothesis testing for Python — powered by LLMs or pure regex
   ```

---

## 2. Add Website URL

In the same "Edit repository details" panel, set the **Website** field to:

```
https://hypotestx.readthedocs.io
```

---

## 3. Add Topic Tags

In the same panel, click **"Manage topics"** and add the following tags one
by one (GitHub calls them "topics"):

```
statistics
hypothesis-testing
natural-language-processing
data-science
python
scipy
llm
pandas
anova
t-test
correlation
chi-square
statistical-testing
machine-learning
```

Click **"Save changes"** when done.

---

## 4. Verify

After saving, the repository's "About" sidebar should show:

- **Description:** Natural language hypothesis testing for Python — powered by LLMs or pure regex
- **Website:** https://hypotestx.readthedocs.io
- **Topics:** statistics · hypothesis-testing · natural-language-processing · data-science · python · scipy · llm · pandas · anova · t-test · correlation · chi-square · statistical-testing · machine-learning

---

## 5. Optional: Add Social Preview Image

On the repository page, go to **Settings → General → Social preview** and
upload a banner image (1280×640 px recommended) that includes the project name
and tagline. This image appears when the link is shared on social media.
